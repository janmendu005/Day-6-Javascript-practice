# To-Do App

A clean and responsive To-Do application built using HTML, CSS, and JavaScript that helps users efficiently manage daily tasks. The application allows users to add new tasks, mark completed tasks, and remove unnecessary items through an interactive and user-friendly interface.

This project demonstrates core front-end development concepts such as DOM manipulation, event handling, dynamic content creation, and responsive UI design. It serves as a practical exercise for understanding how JavaScript interacts with HTML and CSS to create real-world web applications.

## Key Features

* Add new tasks instantly
* Mark tasks as completed
* Delete tasks when no longer needed
* Dynamic updates without page reloads
* Clean and responsive design
* Beginner-friendly code structure

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

## Purpose

This project was developed to strengthen foundational JavaScript skills, particularly in working with the DOM, handling user interactions, and building interactive web applications from scratch.

## Future Enhancements
